## Creating Button 

```luau
local Button = Tab:Button({
    Title = "Button",
    Desc = "Test Button",
    Locked = false,
    Callback = function()
        print("clicked")
    end
})
```

### Set Title
    ```luau
    Button:SetTitle("Title Example")
    ```

### Set Description
    ```luau
    Button:SetDesc("Description Example")
    ```

### Lock Element
    ```luau
    Button:Lock()
    ```

### Unlock Element
    ```luau
    Button:Unlock()
    ```

### Destroy Element
    ```luau
    Button:Destroy()
    ```
