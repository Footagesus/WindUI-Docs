## Load methods

- Load latest version <span className="px-2 py-1 dark:bg-zinc-500 dark:text-zinc-100 bg-zinc-300 text-zinc-800 rounded-full text-sm font-semibold">Recommended</span>
    ```luau
    local WindUI = loadstring(game:HttpGet("https://github.com/Footagesus/WindUI/releases/latest/download/main.lua"))()
    ```

- Load selected version
    ```luau
    local Version = "1.6.41"
    local WindUI = loadstring(game:HttpGet("https://github.com/Footagesus/WindUI/releases/download/" .. Version .. "/main.lua"))()
    ```

            ### Set Notification Lower
                ```luau
                WindUI:SetNotificationLower(true)
                ```
            
            ### Set Font
                ```luau
                WindUI:SetFont("rbxassetid://font-id-here")
                ```
            
            ### Set Theme
                ```luau
                WindUI:SetTheme("Dark")
                ```

            ### Add Theme
                ```luau
                WindUI:AddTheme({
                    Name = "Dark",
                    Accent = "#18181b",
                    Dialog = "#18181b",
                    Outline = "#FFFFFF",
                    Text = "#FFFFFF",
                    Placeholder = "#999999",
                ```
