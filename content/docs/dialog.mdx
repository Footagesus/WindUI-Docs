## Creating Dialog

```luau
local Dialog = Window:Dialog({
    Icon = "bird",
    Title = "Dialog Title",
    Content = "Content Text",
    Buttons = {
        {
            Title = "Confirm",
            Callback = function()
                print("Confirmed!")
            end,
        },
        {
            Title = "Cancel",
            Callback = function()
                print("Cancelled!")
            end,
        },
    },
})
```

### Show Dialog
    ```luau
    Dialog:Show()
    ```

### Close Dialog
    ```luau
    Dialog:Close()
    ```
