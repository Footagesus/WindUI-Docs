## Creating Section 

```luau
local Section = Tab:Section({ 
    Title = "Section",
    TextXAlignment = "Left",
    TextSize = 17, -- Default Size
})
```

### Set Title
    ```luau
    Section:SetTitle("New Title Example")
    ```

### Destroy Element
    ```luau
    Section:Destroy()
    ```
    
Now you can create elements inside this section:
```luau tab="Button"
Section:Button({ ... })
```
