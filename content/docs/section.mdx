## Creating Section 

```luau tab="Full example"
local Section = Tab:Section({ 
    Title = "Section",
    Box = false,
    FontWeight = "SemiBold",
    TextTransparency = 0.05,
    TextXAlignment = "Left",
    TextSize = 17, -- Default Size
    Opened = true,
})
```

```luau tab="Short example"
local Section = Tab:Section({ 
    Title = "Section",
})
```

### Set Title
    ```luau
    Section:SetTitle("New Title Example")
    ```

### Destroy Element
    ```luau
    Section:Destroy()
    ```
    
Now you can create elements inside this section. e.g.
```luau tab="Button"
Section:Button({ ... })
```
