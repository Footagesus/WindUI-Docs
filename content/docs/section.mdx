## Creating Section 

```luau
local Section = Tab:Section({ 
    Title = "Section",
    TextXAlignment = "Left",
    TextSize = 17, -- Default Size
})
```

### Set Title
    ```luau
    Section:SetTitle("New Title Example")
    ```

### Destroy Element
    ```luau
    Section:Destroy()
    ```
