    <AccordionItem value="item-2">
        <AccordionTrigger>How to lock the elements?</AccordionTrigger>
        <AccordionContent>
            You can lock Element with `Locked`
            
            Example with [Button](/docs/button):
            ```luau
            local MySuperButton = Tab:Button({
                Title = "Button",
                Locked = true, -- lock the element
                Callback = function() -- callback will not work. element is locked.
                    print("clicked")
                end
            })
            ```
            
            __All elements with callback have a `Locked` thing__
            <br/>
            Also you can lock the element at any time:
            ```luau
            MySuperButton:Lock()
            ```
            
            or unlock:
            ```luau
            MySuperButton:Unlock()
            ```
            
        </AccordionContent>
    </AccordionItem>
    <AccordionItem value="item-3">
        <AccordionTrigger>How to colorize custom icons?</AccordionTrigger>
        <AccordionContent>
            if you use Nebula icons, or your custom vector* icons. 
            <br/>
            Usually WindUI doesn't colorize custom icons to match the theme. 
            <br/>
            - But you can do it with `IconThemed`
                
                `IconThemed` is available in:
                - [Window](/docs/window)
                - [Tab](/docs/tab)
                - [Tab Section](/docs/tabsection)
                - [Dialog](/docs/dialog)
                - [Popup](/docs/popup)
                - [Notification](/docs/notification)
                - [Paragraph](/docs/paragraph)
                
                For example:
                ```luau tab="Window"
                local Window = WindUI:CreateWindow({
                    -- ...
                    
                    Icon = "rbxassetid://..." -- my custom vector* icon        [!code focus]
                    IconThemed = true                                       -- [!code focus]
                })
                ```
                
                ```luau tab="Tab"
                local Tab = Window:Tab({
                    -- ...
                    
                    Icon = "rbxassetid://..." -- my custom vector* icon        [!code focus]
                    IconThemed = true                                       -- [!code focus]
                })
                ```
                
                ```luau tab="Popup"
                local Popup = WindUI:Popup({
                    -- ...
                    
                    Icon = "rbxassetid://..." -- my custom vector* icon        [!code focus]
                    IconThemed = true                                       -- [!code focus]
                })
                
                ```
                
            and others.
        </AccordionContent>
    </AccordionItem>
    <AccordionItem value="item-4">
        <AccordionTrigger>How to disable OpenButton?</AccordionTrigger>
        <AccordionContent>
            You can disable the `OpenButton` if you don't need it.
            
            ```luau
            Window:EditOpenButton({
                Enabled = false
            })
            ```
        </AccordionContent>
    </AccordionItem>
