## Creating Code 

```lua
local Code = Tab:Code({
    Title = "Code",
    Code = [[print("Hello World!")]]
})
```

### Set Code
    ```lua
    Code:SetCode([[print("New code!")]])
    ```

### Destroy Code
    ```lua
    Code:Destroy()
    ```

### `:OnCopy()`
    ```lua
    Code:OnCopy(function()
        print("Code '" .. Code.Title .. '" Copied!")
    end)
    ```
