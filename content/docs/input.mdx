## Creating Input 

```luau
local Input = Tab:Input({
    Title = "Input",
    Desc = "Input Description",
    Value = "Default value",
    InputIcon = "bird",
    Placeholder = "Enter text...",
    Callback = function(input) 
        print("text entered: " .. input)
    end
})
```

- ## Set Title
    ```luau
    Input:SetTitle("Title Example")
    ```

- ## Set Description
    ```luau
    Input:SetDesc("Description Example")
    ```
    
- ## Set Input value
    ```luau
    Input:Set("New value")
    ```

- ## Lock Element
    ```luau
    Input:Lock()
    ```

- ## Unlock Element
    ```luau
    Input:Unlock()
    ```

- ## Destroy Element
    ```luau
    Input:Destroy()
    ```
